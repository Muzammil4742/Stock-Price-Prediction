# Stock-Price-Prediction
Stock Price Prediction (Machine Learning)
Overview

This project develops a machine learning system to predict stock prices based on historical market data. It demonstrates a complete time-series and regression-based workflow, including data preprocessing, feature engineering, model training, evaluation, and prediction.

Tech Stack

Python, NumPy, Pandas, Matplotlib, Seaborn, Scikit-learn, Jupyter Notebook

Dataset

Historical stock market data containing attributes such as open, high, low, close prices, and trading volume.
Target variable: Stock price (closing price)

Models Used

Linear Regression

Support Vector Regression (SVR)

Random Forest Regressor

LSTM (optional, deep learning extension)

Evaluation Metrics

Mean Absolute Error (MAE), Mean Squared Error (MSE), Root Mean Squared Error (RMSE)

How to Run
pip install -r requirements.txt
jupyter notebook
